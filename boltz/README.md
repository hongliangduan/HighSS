# Vendored Boltz runtime

This directory contains the Boltz runtime used internally by HighSS for model loading, differentiable design, input processing, and final Holo/Apo prediction.

It is installed automatically by the root `setup.sh`:

```bash
python -m pip install -e ./boltz --no-deps
```
